# Template Selection Guide

Use `podcast_interview` for host/guest or question-answer conversations.
Use `solo_course` for one main speaker teaching a concept.
Use `gaming_livestream` for gameplay, streamer reactions, challenge completions, and funny failures.
Use `sports_highlights` for match clips, commentary-driven moments, crowd spikes, goals, saves, fouls, and turning points.

The user must confirm the template before transcription planning continues.
